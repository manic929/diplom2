Обучение основам HTML и CSS

Что представляет из себя HTML и CSS?

HTML - Стандартизированный язык разметки документов для просмотра веб-страниц в браузере. Веб-браузеры получают HTML документ от сервера по протоколам HTTP/HTTPS или открывают с локального диска, далее интерпретируют код в интерфейс, который будет отображаться на экране монитора.

CSS (Cascading Style Sheets, каскадные таблицы стилей) — язык описания внешнего вида HTML-документа. Это одна из базовых технологий в современном интернете. Практически ни один сайт не обходится без CSS, поэтому HTML и CSS действуют в единой связке.

Возможности HTML и CSS

Возможности HTML:

  • Позволяет создавать и структурировать разделы, параграфы, заголовки, ссылки и блоки для веб-страниц.
  • HTML-документ можно создать в любом редакторе, который есть в операционной системе.
  • Позволяет создавать таблицы, различные виды списков, добавлять изображения, создавать формы для взаимодействия с пользователем.

Возможности CSS:

  • Осуществлять выбор элемента или группы элементов, к которым будет применен тот или иной стиль.
  • Осуществлять создание блоков: устанавливать размеры самих блоков, отступов, полей и рамок.
  • Осуществлять позиционирование блоков.
  • Управлять шрифтом на странице.
  • Управлять фоном на странице.

Как создать HTML-документ?

Для того, чтобы создать HTML-документ необходимо:

  1. Открыть любой текстовый редактор (Например, блокнот)
  2. Разметить HTML основными тегами и набрать произвольный текст (также разметив его тегами)
  3. Сохранить файл с расширением .html
